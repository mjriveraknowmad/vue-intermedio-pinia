Créditos:

Curso de Vue Intermedio https://udemy.com/course/vue-intermedio/learn/lecture/34483924



### Dev
Pasos

1. Clonar el proyecto
2. Ejecutar
```
npm install
```

3. Levantar el backend server
```
npm run backend-server
```

4. Renombrar .env.template por .env

5. Ejecutar la aplicación de Vue.js
```
npm run dev
```
